# Teslo Shop

## Configuración

1. Clona el repositorio.
2. Clona el archivo `.env.template` y renombralo a `.env`. Si lo necesitas, renombra las variables de entorno.
3. Ejecuta `npm install` en la terminal para instalar las dependencias.
4. Comprueba que el back-end de Teslo Shop ([puedes obtenerlo aquí](https://github.com/Klerith/nest-teslo-shop/tree/complete-backend-paginated-filters)) esté corriendo en el puerto `3000`, o el que configuraste en el archivo `.env`.
5. Ejecuta `npm run dev` para arrancar el servidor Vite de desarrollo.
